# homebrew-flowlyt

Homebrew tap for [Flowlyt](https://github.com/harekrishnarai/flowlyt) — a multi-platform CI/CD security analyzer.

```bash
brew install harekrishnarai/flowlyt/flowlyt
# or
brew tap harekrishnarai/flowlyt
brew install flowlyt
```

The `Formula/flowlyt.rb` is updated automatically by Flowlyt's release workflow.
